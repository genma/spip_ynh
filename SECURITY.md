# Version française

**Maintenance et signalement des failles de sécurité pour SPIP (package Yunohost)**

Les annonces de maintenance et de sécurité sont publiées par l’équipe SPIP sur la page dédiée : https://blog.spip.net/_L-equipe-maintenance_.html

**Intégration des correctifs**
Nous nous efforçons d’intégrer rapidement les mises à jour et correctifs de sécurité dans le package SPIP pour Yunohost.
Si une version corrective n’est pas encore intégrée, vous pouvez ouvrir une issue sur le dépôt du package.

**Signalement d’une faille de sécurité**
Pour signaler une faille de sécurité à l’équipe SPIP, envoyez un email à : securite@spip.net
Ne le faites pas publiquement, ni dans un ticket, ni sur un forum, afin de préserver la sécurité de tous les utilisateurs.

Merci pour votre contribution !

# English version

**Maintenance and Security Reporting for SPIP (Yunohost Package)**

Maintenance and security announcements are published by the SPIP team on the dedicated page: https://blog.spip.net/_L-equipe-maintenance_.html

**Integration of Security Patches**

We strive to quickly integrate security updates and patches into the SPIP package for Yunohost.
If a corrective version has not yet been integrated, you can open an issue on the package repository.

**Reporting a Security Vulnerability**

To report a security vulnerability to the SPIP team, please send an email to: securite@spip.net

Do not report it publicly, in a ticket, or on a forum, in order to preserve the security of all users.spip.net

Thank you for your contribution!
